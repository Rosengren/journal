# Journal

**Create Project Command**
    spring init --boot-version=1.3.1.RELEASE --build=gradle --groupId=com --name=Journal

**Build Command**
    gradle build

**Clean Build**
    gradle clean build

**Run Server**
    gradle bootRun


**Software Versions**

* Java 8
* Gradle 2.14.1
* Spring Boot CLI 1.3.2
